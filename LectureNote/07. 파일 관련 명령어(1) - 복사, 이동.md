# 7강 파일 관련 명령어(1) - 복사, 이동

### 파일/디렉터리의 복사(copy)

```bash
/bin/cp [-options] <scr> <dest>
```

- 주고 옵션
    - -i: interactive
    - -r: recursive

| 사용 형식 | 동작 |
| --- | --- |
| cp 파일1 파일2 | 파일 1을 파일 2로 복사 |
| cp 파일들 디렉토리 | 파일들을 디렉토리 밑에 같은 이름으로 복사 |
| cp -r 디렉토리1 디렉토리 2 | 디렉토리1을 디렉토리2로 복사<br>디렉토리1의 파일도 모두 복사됨 |
