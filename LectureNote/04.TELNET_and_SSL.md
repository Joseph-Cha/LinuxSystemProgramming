# 4강 TELNET / SSL

## 텔넷(Telnet)

### 원격 접속 서비스

- 원격지 컴퓨터에 연결하여 그 컴퓨터의 자원을 활용하는 서비스
- 포트번호: 기본 23번
- 키보드와 모니터는 로컬에 있는 것을 사용하되 컴퓨터 자원은 원격지 서버가 제공해 주는 것을 사용한다.

### 서비스를 이용하기 위해서는

- 텔넷 서버와 텟넷 클라이언트 필요
- 접속하고자 하는 컴퓨터의 사용자 계정을 발급 필요
    - 보통 UNIX 시스템에 접속
    - 아이디: 비밀번호

## SSH(Secure SHELL)

- 텔넷처럼 원격 접속을 지원하되 텔넷과 달리 모든 통신 내용을 암호화 함
    - 포트번호: 기본 22번
- 초최 접속시 인증키 생성 여부를 묻는 질문에 yes 선택
